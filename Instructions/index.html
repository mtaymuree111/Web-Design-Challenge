Overall Summary - Latitude vs Weather Attributes

In this project, weather varaitions as a realtion of equatorial distance were examined; specfically how does close proximity to the equator impact weather patterns. Data was sourced from OpenWeatherMap API and calling criteria in from approximately 500 cities.

Once the API call was complete, using the coding language of Matplotlib, different weather attributes were plotted (ie wind speed, humidity, temperature and cloudiness) in relationship to latitude. Please nagivate using the bar above to access the plots, data and maps.
